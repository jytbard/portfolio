Architecting data warehouse to compare subscription plans

Overview:

This project aimed to design and build a data warehouse system for comparing subscription prices across multiple providers, offering insightful data to users for a platform called Bill Blitz.

Roles & Responsibilities:

Designing a comprehensive data model and appropriate schema using Snowflake.

Developing ETL pipelines in Azure Data Factory (ADF) to extract, transform, and load data from various sources, ensuring data accuracy and consistency.

Integrating external data sources, performing deduplication, and validating data to maintain high-quality standards.

Optimizing data retrieval and performance through indexing, partitioning, and query optimization.

Ensuring data security by implementing encryption, access control, and compliance with regulatory requirements.

Creating Power BI dashboards integrated with Snowflake for enhanced data analysis and reporting.

Providing detailed documentation covering the entire implementation process.
